# flask-line-bot

以聖經當作題材，實作Line Bot．

### Environment Setup
```
export FLASK_APP=flb.py
export CHANNEL_ACCESS_TOKEN=<CHANNEL_ACCESS_TOKEN>
export CHANNEL_SECRET=<CHANNEL_SECRET>

flask run
```

### Usage
```
> (space)Ps 23
< 23:1 ...
```
